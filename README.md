# Payal Dhumal Portfolio

A modern full-stack developer portfolio website highlighting professional summary, technical skills, internship experience, and featured projects.

## Description

This portfolio is designed to present recruiter-ready information with a clean and responsive UI. It includes:

- Professional introduction and role focus
- Technical skills grouped by frontend, backend, databases, and tools
- Internship experience
- Dynamic project section with GitHub-based recent project ranking
- Featured professional resume project
- Contact section with LinkedIn and GitHub

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Project Structure

- `index.html` - Main portfolio page
- `css/style.css` - Styling and layout
- `Images/` - Project and profile assets
- `PayalDhumal_Resume.pdf` - Resume file

## Run Locally

```powershell
python -m http.server 5500
```

Open `http://localhost:5500/index.html` in your browser.
